# 🚀 Obilet QA Case Study

Bu proje, **https://www.obilet.com** sitesi üzerinde verilen senaryoya uygun olarak gerçekleştirilen **test otomasyon çalışmalarını** içermektedir.  
Testler, **Selenium WebDriver** kullanılarak **Java** diliyle yazılmıştır ve kişisel olarak geliştirdiğim **custom test otomasyon frameworkü** kullanılmıştır 🧪⚙️

---

## 📝 Test Senaryosu

Aşağıdaki adımlar otomasyon kapsamında test edilmiştir:

## 📝 Test Case 1

1. 🌐 https://www.obilet.com sitesi ziyaret edilir.  
2. 🔍 Kullanıcı ana menüden otobüs dikeyine tıklar.
3. 📋 İstenilen 2 il için kalkış ve varış seferi seçilir. 
4. 📋 İstenilen gidiş tarihi seçilir ve arama yapılır
5. 🔍 Sefer listeleme ekranından istenilen bir sefer seçilir.
6. 🔢 Koltuk seçim kurallarına dikkat edilerek istenilen cinsiyette koltuk seçimi yapılır
7. ✅ Ödeme sayfasında doğru sefer ve doğru fiyatın seçildiği eşleştirilir. 

---


## 📝 Test Case 2

1. 🌐 https://www.obilet.com sitesi ziyaret edilir.  
2. 🔍 Kullanıcı ana menüden otel dikeyine tıklar.
3. 📋 Otel için istenilen bir lokasyon seçilir. 
4. 📋 Otel aramak için butona tıklanır.
5. 🔍 Çıkan sonuçlar içerisinde "Yarım Pansiyon" oteller filtrelenir.
6. 🔢 Sonuçların doğru şekilde filtrelendiği görülür.
7. 🔍 Sonuçlar düşükten yükseğe fiyat ile sıralanır.
8. ✅ Sonuçların doğru şekilde sıralandığı görülür.




## 🛠️ Kullanılan Teknolojiler

- ☕ **Programlama Dili:** Java  
- 🤖 **Test Otomasyon Aracı:** Selenium WebDriver  
- 🧪 **Test Framework:** TestNG  
- 🌍 **Tarayıcı:** Google Chrome
